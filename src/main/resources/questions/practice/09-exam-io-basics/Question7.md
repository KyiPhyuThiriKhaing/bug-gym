## Question 7
Read an `R x C` matrix and print the column sums.

Input format:
- First line: `R C`
- Next `R` lines: `C` integers each

Output column sums separated by spaces.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2 3 <br> 1 2 3 <br> 4 5 6|5 7 9|
|3 2 <br> 1 0 <br> 2 2 <br> 3 5|6 7|
