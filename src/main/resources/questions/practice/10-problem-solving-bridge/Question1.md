## Question 1
Read N integers and print `Balanced` if the sum of the first half equals the sum of the second half, else print `Not Balanced`.

If N is odd, ignore the middle element.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|6 <br> 1 2 3 3 2 1|Balanced|
|5 <br> 2 1 9 1 2|Balanced|
|4 <br> 1 2 3 4|Not Balanced|
