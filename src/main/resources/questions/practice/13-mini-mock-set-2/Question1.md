## Question 1
Read N integers and print the longest contiguous segment where values are non-decreasing.

Output format: `startIndex endIndex length` (0-based indexes).

If multiple segments have the same length, print the earliest one.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|8 <br> 1 2 2 1 3 4 0 0|0 2 3|
|5 <br> 5 4 3 2 1|0 0 1|
