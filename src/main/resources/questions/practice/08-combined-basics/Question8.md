## Question 8
Read a matrix and print the sum of each row.

Input format:
- First line: `R C`
- Next `R` lines: `C` integers each

Output the row sums separated by spaces.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2 3 <br> 1 2 3 <br> 4 5 6|6 15|
|3 2 <br> 5 5 <br> 1 2 <br> 0 7|10 3 7|
