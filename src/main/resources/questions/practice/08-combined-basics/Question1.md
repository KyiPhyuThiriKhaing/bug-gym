## Question 1
Count how many numbers are positive, negative, and zero.

The input starts with integer N, followed by N integers.

Output in this exact format: `positive negative zero`.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|6 <br> -2 0 5 0 9 -1|2 2 2|
|4 <br> 1 2 3 4|4 0 0|
|3 <br> 0 0 0|0 0 3|
