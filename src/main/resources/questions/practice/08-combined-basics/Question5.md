## Question 5
Read a sentence and count how many vowels it contains.

Consider vowels: `a, e, i, o, u` (both lowercase and uppercase).

The input is a single line sentence. Output the vowel count.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|Hello World|3|
|BUG GYM|1|
|sky|0|
