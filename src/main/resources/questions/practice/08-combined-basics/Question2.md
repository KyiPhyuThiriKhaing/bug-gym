## Question 2
Read N numbers and print the running maximum after each number.

The input starts with integer N, followed by N integers.

Print the running maximum values separated by spaces.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 2 7 4 9 1|2 7 7 9 9|
|4 <br> -3 -5 -1 -8|-3 -3 -1 -1|
