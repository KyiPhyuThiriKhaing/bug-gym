## Question 6
Write a program that defines a method `sumArray(int[] arr)` and prints the sum of all elements.

The input starts with integer N, followed by N integers. The output is the sum.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 1 2 3 4 5|15|
|4 <br> -2 8 -1 5|10|
|1 <br> 42|42|
