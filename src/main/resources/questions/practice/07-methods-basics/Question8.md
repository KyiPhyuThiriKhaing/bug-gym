## Question 8
Write a program that defines a method `toFahrenheit(double c)` and prints the Fahrenheit value.

Use the formula: `F = (C * 9 / 5) + 32`.

Print the result rounded to 1 decimal place.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|0|32.0|
|25|77.0|
|-40|-40.0|
