## Question 7
Write a program that defines a method `containsChar(String s, char target)` and prints `Found` if the character exists in the string, otherwise prints `Not Found`.

The input has a string and then a character. Output `Found` or `Not Found`.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|banana a|Found|
|java z|Not Found|
|HELLO H|Found|
